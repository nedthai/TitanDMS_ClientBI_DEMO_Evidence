**Dealer Comprehensive Invision** is an advanced Business Intelligence (BI) reporting system tailored for automotive dealers. This project provides a high-quality, real-time data visualization ecosystem designed to help dealers extract valuable insights and make informed, rational, and data-driven decisions.

## 🚀 Key Project Highlights

### 1. Comprehensive & High-Quality Reporting
The system provides multi-dimensional views of dealership performance, ensuring that stakeholders have a clear "invision" of their operations:
*   **Vehicle Aged Stock**: Real-time snapshot of inventory with drill-down capabilities by Company, Location, and Vehicle Class to optimize stock turnover.
*   **Vehicle Sales Volume**: Detailed historical analysis of sales network performance across different regions and timeframes.
*   **Operational Transparency**: Streamlined dashboards that prioritize clarity and actionable intelligence over raw data.

### 2. AI & Predictive Analytics
This project goes beyond standard reporting by integrating advanced technology and mathematical rigor:
*   **AI-Enhanced Development**: Developed using **Google Antigravity**, leveraging AI for sophisticated "code vibe," premium UI/UX design, and rapid architecture implementation.
*   **Forecasting Models**: Implemented **Simple Linear Regression** using **Ordinary Least Squares (OLS)** to predict sales volume for the next 6-10 months.
*   **Mathematical Validation**: The predictive engine provides statistical metrics such as **Coefficient of Determination (R²)** and **Pearson Correlation (r)** to ensure forecast reliability and trend strength analysis.

## 🛠️ Technology Stack
*   **[Evidence.dev](https://evidence.dev/)**: The core BI framework for creating high-performance, code-based reports.
*   **DuckDB**: Leveraged for ultra-fast, in-process analytical processing and complex SQL-based mathematical modeling.
*   **Python**: Used for data orchestration and ETL processes from SQL Server to the analytical layer.
*   **Tailwind CSS**: Custom premium styling for a modern, "glassmorphism" inspired dashboard aesthetic.

## 🚦 Getting Started

### Prerequisites
*   Node.js (v18 or later)
*   Python 3.x (for data sourcing)

### Installation
1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Pull Data (ETL)**:
    ```bash
    npm run sources
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
